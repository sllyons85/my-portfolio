IMDb Incremental ETL Pipeline

This project is a professional-grade incremental ETL pipeline for IMDb datasets. It demonstrates skills in data engineering, automation, and database design using Python and MySQL.

Features

  • Incremental sync using row hashing
  • Genre normalization into relational tables
  • Indexing for query performance
  • Logging and backup system

Project Structure

  • new_download/: Raw IMDb files
  • new_extract/: Extracted TSVs
  • previously_uploaded/: Reference snapshot
  • logs/imdb_sync_logs.txt: Operation log
  • imdb_etl_script.py: Core ETL script

Workflow

  1. Download and extract IMDb data
  2. Compare against previously uploaded data
  3. Insert new/changed records using SQL
  4. Normalize genres into a relational structure
  5. Create indexes
  6. Backup current dataset

How to Run

pip install pandas sqlalchemy tqdm pymysql
python imdb_etl_script.py

Technologies

  • Python (pandas, SQLAlchemy, tqdm)
  • MySQL
  • Logging, row hashing, genre parsing
